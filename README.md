## Installing

You can execute a wizard tool using:

```
.bash/install
```

To populate your database use:

```
rake db:fetch
```

To drop your database (and character images) use:

```
rake db:drop
```

To start the development server use:

```
rake develop
```

To start the assets development watcher use:

```
grunt develop
```
